# minerproxy
eth挖矿代理项目，无任何抽水，开源，仅供学习，需要什么功能自己改
**本项目支持sock5代理挖矿**

**执行方案:**
* 需要一个pfx（带密码的证书，不懂怎么弄自己去百度openssl生成pfx文件，放在同一目录，取名为"cert.pfx"，密码为123456,若有需要改动可到EthRoute.cs修改）
* 下载相关方案然后运行
* 如果有自定义端口可加 --urls http://*:3333 此处为3333端口（web管理端）
* 执行完毕后打开相关页面即可管理
